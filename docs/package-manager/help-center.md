# Local Help Center

The local Help Center belongs in `lemonade-admin`, the dedicated internal admin package.

## What users see

- Operator help for POS, inventory, daily close, and reports.
- Admin help for packages, backups, updates, and internal access.
- Troubleshooting steps for common offline failures.

## What is separated

Developer and maintainer details are separated into a maintainer section and are not the default view for attendants.

## Source of truth

Markdown docs in this repo are version-controlled source material. `lemonade-admin` renders the operator/admin subset inside the local internal website.
